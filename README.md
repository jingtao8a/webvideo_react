# WebVideo的功能设计思路
## 软件需要实现的功能
> 1.用户登入
> 2.登入后可以查看对应的文件系统层级，以及视频和图片信息

## 前端部分
1. 需要一个登入页面
    >登入页面需要输入用户名、密码信息
2. 登入成功后展示的页面需要一个侧边导航栏和一个面包屑
   > 根据服务器资源具体的目录信息展示对应的嵌套导航栏，点击对应的导航栏选项，面包屑也同步对应的文件路径变化
3. 右方上侧需要一个导航栏用作分类选择 文件 视频 图像，右方下侧有对应的分页条
   > 并且右侧的内容部分展示对应的内容
